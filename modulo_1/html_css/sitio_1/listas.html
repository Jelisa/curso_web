    URL localizador Universal de Recursos.
    Es la dirección web (o local) donde se encuentra un recurso (documento, imagen, video, web ...)
    Hyperenlace (aka también "enlace", "vínculo", "link", "hipertexto")
    Es un texto o imagen sobre al que pulsar nos envía a otro sitio web.
